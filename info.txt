this is a repository for a lab project regarding real time data processing using kafka and spark. The project is based on the following architecture:

![image](https://github.com/alexandru-ionescu/kafka-spark/assets/113468433/63753827-693a-4288-9539-4248483b154e)

The project is divided into 3 parts:

1. **Data Producer** - a python script that generates random data and sends it to a kafka topic
2. **Data Consumer** - a python script that consumes data from a kafka topic and sends it to a spark streaming application
3. **Data Processing** - a spark streaming application that processes the data and sends it to a kafka topic

## Data Producer

The data producer is a python script that generates random data and sends it to a kafka topic. The data is generated using the `faker` library and the data is sent to a kafka topic using the `kafka-python` library. The data is sent to the kafka topic in a json format.
